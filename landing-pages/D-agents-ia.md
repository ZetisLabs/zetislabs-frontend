## PROJET PHARE

### Titre

**Mail-Writer — Notre agent IA open-source**

### Contenu

Mail-Writer est un agent IA qui rédige des emails en reproduisant votre style d'écriture.

**Comment ça marche** :

1. L'agent analyse vos emails existants
2. Il apprend votre ton, vos formulations, vos signatures
3. Il génère des emails qui sonnent comme vous

**Résultat** : Plus besoin de réécrire les mêmes emails. L'agent le fait pour vous.

`Voir sur GitHub →` https://github.com/ZetisLabs/Mail-Writer
